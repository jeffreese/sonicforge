### FX size note

Riser WAV files are large (500 KB–6.3 MB each). Before committing, transcode to OGG @ 128 kbps to shrink 5–10x:

```bash
ffmpeg -i input.wav -c:a libvorbis -q:a 4 output.ogg
```

Expected post-transcode sizes: 50 KB–600 KB per file.

## Cropping notes

Two snare samples (CVLTIV8R, Ganon) are multi-hit reels rather than single transients. Before committing, each will be cropped to a single representative hit using ffmpeg:

```bash
# Example — adjust -ss start and -t duration per file after inspecting waveforms
ffmpeg -i input.wav -ss 0.0 -t 0.3 -c copy first_hit.wav
```

Target file sizes after cropping: ~50–150 KB each (matching the single-hit snares already in the set).
